Steps to Setup Front-End on Local System
    - Take Clone from the Git
    - Then run Commands
        - npm install
        - npm start
        - In src\base_url\config.js file change base_url with your back-end URL


Steps To Steup Back-end (Laravel) on Local System
    - Take Clone from Git
    Then Commands to Run the Project on Local

     - composer update
     - Run cp .env.example .env or copy .env.example .env. and Change your Database credentials in the File
     - php artisan key:generate
     - php artisan migrate
     - php artisan serve
